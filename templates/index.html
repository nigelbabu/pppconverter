About

This application was built as a solution to one of the challenges in Get the Data Challenge. The code is open source and MIT licensed and can be found on GitHub.

The data used by this application comes from The World Bank. The data was then converted into a simplified table by taking the latest available data for every country. The source data used by this application is on GitHub along with the code (Last Updated: Mar 2018). Favicon by Arthur Shlain on The Noun Project.
